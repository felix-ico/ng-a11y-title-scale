## Installation
Firstly you need `ng cli`, `Node.js` and `NPM` in order to run this app locally

1. Clone/download the repository
2. Open the terminal in project root.
3. Run `npm install`
4. Run `ng serve` 
5. Navigate to `http://localhost:4200/`
